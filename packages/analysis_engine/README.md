# 分析引擎

`analysis-engine` 是一个确定性数据分析引擎，负责将非标准化的原始记录（如账单流水、健康数据等）清洗、聚合，并检测异常，最终生成面向 LLM 友好的事实简报。

## 能力边界

该包是一个基础的分析底座，遵循“让数学的归数学，让文学的归 AI”的原则：
- **无状态计算**：不持有任何数据库连接或环境配置。
- **确定性输出**：所有的统计（总计、环比、分组聚合）和异常检测均基于确定的数学逻辑，不依赖 LLM。
- **领域无关**：通过 `inflow` / `outflow` / `neutral` 的基础抽象，支持财务账单、工程预算甚至运动指标。

如果原始数据**没有 direction 字段**，`analysis_engine` 默认把正数金额视为 outflow (流出)，以兼容常见的“全为正数的支出账单”。负数同样视为 outflow。
